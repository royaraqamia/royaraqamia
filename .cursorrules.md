# EXPERT DEVELOPER & DESIGNER INSTRUCTIONS

## 1. TECH STACK (The "Golden Stack")
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS + Shadcn/UI (Radix UI) + Lucide Icons
- **AI/Agents:** Vercel AI SDK (for streaming and tool calling)
- **Language:** TypeScript (Strict mode)

## 2. DESIGN & UI RULES (Critical)
- **Look & Feel:** Modern, clean, "Linear-style" or "Apple-style" aesthetics.
- **Components:** ALWAYS use existing Shadcn components from `@components/ui` before creating new ones.
- **Spacing:** Use Tailwind spacing (p-4, m-8, gap-4). Never use arbitrary pixels (e.g., w-[123px]).
- **Colors:** Use CSS variables defined in globals.css (bg-background, text-foreground).
- **Responsive:** Mobile-first approach. Always check `md:` and `lg:` breakpoints.
- **Animation:** Use `framer-motion` for complex animations, or `tailwindcss-animate` for simple transitions.

## 3. CODING STANDARDS
- **Functional:** Use React Server Components (RSC) by default. Use 'use client' only when interaction (state/hooks) is needed.
- **Clean Code:** DRY principles. Abstract logic into hooks (`/hooks`).
- **AI Agents:** When building agents, use `ai/rsc` for streaming UI and `zod` for structured outputs.

## 4. BEHAVIOR
- When asked to build a UI, look for a matching Shadcn component first.
- If the design is complex, break it down into smaller sub-components.
- Do not explain basic concepts. Just write the code.
